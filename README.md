# grapho

GraphicsApi Object.
Header only OpenGL3 / DX11 mini wrapper.

using `c++2b` std::expected for error handling.

- msvc17
- clang-16
- gcc-12

## namespace

- `grapho::gl3`
- `grapho::dx11`
- `grapho::imgui`
